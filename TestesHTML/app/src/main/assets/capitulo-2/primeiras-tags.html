Títulos com TAGs de heading

As tags de heading são tags de conteúdo e vão de <h1> a <h6>, seguindo a ordem de importância. Por ser uma tag de conteúdo deve se fechar a tag.

As ferramentas de indexação de conteúdo para buscas, como Google e Yahoo! levam em consideração essa ordem e relevância.

Parágrafos

Para colocar textos no body eles devem sempre estar dentro de alguma tag a mais indicada para textos comuns é a tag de parágrados <p>.

Para marcar ênfase em trechos do texto pode se utilizar marcações de ênfase. Pode deixar o texto mais forte com tag <strong> ou deixar o texto com uma ênfase mais acentuada com a tag <em>. Também há a tag <small> que diminui o tamanho do texto.

Por padrão os navegadores renderizam a tag <strong> em negrito e a tag <em> em itálico. Existem as tag <b> e <i> que atingem o mesmo resultado visualmente, porem as tags <strong> e <em> definem mais significado ao conteudo.

Imagens

A tag <img> define uma imagem em uma pagina HTML e necessita de dois atributos preenchidos src e alt. O primeiro aponta para o local da imagem e o segundo um texto alternativo para a imagem caso essa não possa ser carregada.

No HTML 5 existem duas novas tags para especificar imagens <figure> e <figcaption>. A tag <figure> define uma imagem com a tag <img> e permite adicionar uma legenda para imagem com a tag <figcaption>.
