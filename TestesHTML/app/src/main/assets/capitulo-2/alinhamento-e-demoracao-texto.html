Propriedade que refere ao alinhamento do texto: text-align No exemplo:

p {
text-align: right;
}

O alinhamento anterior é para alinhar o texto a direita(right). Também é possível determinar que um elemento tenha seu conteúdo alinhado ao centro(center) , ou ocupar toda a largura do elemento aumentando o espaçamento entre as palavras (justify). O padrão porem é alinhasr os textos a esquerda(left).

O estilo definido no css tambem altera outras propriedades como:
A altura de cada linha com o atributo: line-height: [valor]px.
O espaçamento entre cada letra com o atributo: letter-spacing: [valor]px.
O espaçamento entre cada palavra com o atributo: word-spacing: [valor]px.
O tamanho da margem da primeira letra do texto com o atributo: text-indent: [valor]px.

Bordas

As propriedades do CSS para definirmos as bordas de um elemento nos apresentam uma serie de opções.
Para cada borda de um elemento podemos determinar sua cor, seu estilo de exibição e sua largura. Por exemplo:

body {
border-color: red;
border-style: solid;
border-width: 1px;
}

Para que o efeito da cor sobre a borda surta efeito, é necessário que a propriedade border-style tenha qualquer valor diferente do padrão none.
