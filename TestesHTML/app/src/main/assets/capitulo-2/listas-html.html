Listas em HTML

Para fazer uma lista não ordenada uma a tag <ul> e dentro dela cada item da lista é delimitado por uma tag <li> Pode haver listas aninhadas dentro de uma lista.
Lista não ordenada:

  • Primeiro item da lista
  • Segundo item da lista
  • Terceiro item da lista:
    • Primeiro item da lista aninhada
    • Segundo item da lista aninhada
    • Terceiro item da lista aninhada
  • Quarto item da lista
    1. Primeiro item da lista aninhada ordenada
    2. Segundo item da lista aninhada ordenada
    3. Terceiro item da lista aninhada ordenada
  • Quinto item da lista

Para fazer uma lista ordenada uma a tag <ol> e dentro dela cada item da lista é delimitado por uma tag <li> Pode haver listas aninhadas dentro de uma lista.
Lista ordenada:

  1. Primeiro item da lista
  2. Segundo item da lista
  3. Terceiro item da lista:
    • Primeiro item da lista aninhada
    • Segundo item da lista aninhada
    • Terceiro item da lista aninhada
  4. Quarto item da lista
    1. Primeiro item da lista aninhada ordenada
    2. Segundo item da lista aninhada ordenada
    3. Terceiro item da lista aninhada ordenada
  5. Quinto item da lista

A lista é iniciada com tag <dl> e cada termo de definição está em uma tag <dt> e cada definição de termo esta entre a tag <dd>
