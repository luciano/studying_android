Primeira pagina em HTML, feita no Android.

As paginas HTML tem que iniciar com a tag <html> e possuir duas tags filhas <head> e <body> e nelas vão ser colocados os componentes para compor a tela e marcar o texto de modo a ficar agradável ao usuário.

Nessa pagina tem o texto com tag <title> para dar titulo a pagina dentro da tag principal <head> que define coisas que são de interesse do navegador, como onde está a folha de estilo e também qual encoding o navegador vai usar, o padrao é "utf-8" também chamado de Unicode.

A tag <body> contém o corpo do documento, o que deve ser exibido pelo navegador. É necessario que o body tenha ao menos um filho, um tag HTML dentro dele.

O <!DOCTYPE html> não é uma tag HTML porem ele indica ao navegador qual versao do HTML ele deve usar. Usar o DOCTYPE simples informa ao navegador para usar a ultima versão do HTML, no caso o HTML 5.
