Usando CSS

Antigamenta para customizar o design dos elementos da pagina devia fazer isso no proprio HTML. Porem isso é má prática hoje e jamais deve ser usado.
Exemplo de texto em azul e vermelho usando <font> no próprio HTML, é má prática fazer isso.

Para fazer isso surgiu o CSS que é outra linguagem separada do HTML com objetivo unico de cuidar da estilização da pagina.

Sintaxe do CSS

A sintaxe do CSS tem estrutura simples: é declaração de propriedades e valores por um sinal de dois pontos ":" e cada propriedade é separada por um sinal de ponto e virgula ";" conforme o exemplo:

background-color: yellow;
color: blue;

O elemento que receber essas propriedade será exibido com texto na cor azul e fundo amarelo.

Essas propriedades do CSS podem ser declaradas de três maneiras distintas

Atributo style

O atributo style pode ser colocado no próprio elemento. Exemplo:

<p style="color: blue; background-color:yellow;">
O contúdo desta tag será exibido em azul com fundo amarelo no navegador!
</p>


Colocando em prática:

O contúdo desta tag será exibido em azul com fundo amarelo no navegador!

Por ser colocada também dentro do HTML esse modo deve ser evitado.

A TAG style

Podemos declarar as propriedades dentro da tag style. E precisamos indicar de alguma maneira a qual elemento nos referimos. Fazemos isso utilizando um seletor CSS. É basicamente uma forma de buscar certos elementos dentro da página que receberão as regras visuais que queremos.

No exemplo usaremos o seletor de h3 para nao quebrar as tags p feitas anteriormnete e serão definidos no <head>:

<style>
h3 {
background-color: yellow;
color: blue;
}
</style>

O código anterior da tag <style> indica que estamos alterando a cor e o fundo de todos os elementos com tag h3. Dizemos que selecionamos esses elementos pelo nome de sua tag, e aplicamos certas propriedades CSS apenas neles.
