Trasnformação de texto

A propriedade text-transform permite realizar transformações em textos e seus possíveis valores são:

  • uppercase - Todas as letras em maiúsculo.
  • lowercase - Todas as letras em minusculo.
  • capitalize - Só as primeiras letras das palavras em maiúsculo.

Exemplo:

Exemplo de frase usando transformação de texto. text-transform: uppercase

Exemplo de frase usando transformação de texto. text-transform: lowercase

Exemplo de frase usando transformação de texto. text-transform: capitalize

Decoração de Texto

Existe uma serie de decorações que o navegador coloca no texto dependensdo das tags que utilizamos. Exemplos:
link sublinhado del riscado

Mas é possível mudar a decoração de um texto ou até mesmo tira-la. Com a propriedade text-decoration. E pode receber os valores:
none: nenhuma decoração;
underline: linha embaixo do texto
overline: linha em cima do texto
line-through: linha no meio do texto
blink: texto fica piscando na tela

Exemplo de texto usando a decoração. text-decoration: none;

Exemplo de texto usando a decoração. text-decoration: underline;

Exemplo de texto usando a decoração. text-decoration: overline;

Exemplo de texto usando a decoração. text-decoration: line-through;

Exemplo de texto usando a decoração. text-decoration: blink;

Extra

O valor inherit indica para o elemento filho que ele deve utilizar o mesmo valor presente no elemento pai, sendo assim, toda vez que o tamanho do elemento pai for alterado, automaticamente o elemento filho herdará o novo valor, facilitando assim, a manutenção do código.
Lembre-se de que o inherit também afeta propriedades que não são aplicadas em cascata.
As propriedades height e width não são cascateadas, mas essa propriedade força isso.
